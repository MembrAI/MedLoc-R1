# MedLoc-R1
## Status

🚧 The code is currently being organized, cleaned, and further extended for the journal version of this work. We sincerely apologize that the public release will be delayed.

At this stage, the implementation is being actively updated to incorporate additional experiments and refinements for the journal extension. To ensure clarity, reproducibility, and long-term usability, we plan to release a cleaner and more complete version of the code after this process is finished.

We will make the code publicly available as soon as the journal extension is completed. Before the official release, the core methodology and key implementation details are already described in the paper, which we hope can serve as a helpful reference for readers interested in understanding or reproducing our work.
